# Formato das Instâncias

Cada arquivo de instância usa o formato:

1. Primeira linha: `n m`
   `n` = número de pessoas e `m` = tamanho do grupo a selecionar.
2. Demais linhas: `i j a`
   índices `i` e `j` (base 0, com `i < j`) e afinidade `a` entre o par.

Todos os pares são listados uma única vez.
